Ferramentas que possuo experiência e/ou conhecimento

Microsoft Power BI

Trabalho a mais de 4 anos com a principal ferramenta de visualização do mercado e depois de entregar vários projetos, adquiri conhecimento sobre as melhores práticas para uma implementação de sucesso dessa ferramenta, seja ela pequena, média ou enterprise. Possuo certificados oficiais Microsoft: MCSA BI, DA-100 e DP-500.

Apache Hop

O Apache Hop é uma evolução da já excepcional ferramenta open source de ETL Pentaho Data Integration, essa ferramenta low-code, usada na implementação da etapa mais importante de um projeto de BI, acaba sendo mais intuitiva e com diversas melhorias em relação a sua sucessora.

Microsoft Excel

A principal ferramenta de planilhas do mercado. Trabalho há mais de 15 anos com Excel e já desenvolvi diversas soluções com está ferramenta, seja usando o recursos nativos, o suplementos Powers ou mesmo VBA. Possuo alguns certificados oficiais Microsoft dessa ferramenta.

Analysis Services

O Analysis Services é uma ferramenta de Business Intelligence da Microsoft que permite criar e gerenciar modelos de dados multidimensionais e tabulares. Uma das principais funcionalidades do Analysis Services é a criação de uma camada semântica que abstrai a complexidade do modelo de dados e oferece aos usuários finais uma visualização simplificada e intuitiva das informações.

Linguagem SQL

A linguagem SQL (Structured Query Language) é uma das ferramentas mais importantes para um analista de dados. Em minha experiência como analista de Business Intelligence e analista de negócio tenho utilizado constantemente essa linguagem para consulta e contrução de relatórios, seja em bancos on-premisses ou na nuvem.

Linguagem Python

A principal linguagem de programação para análise de dados atualmente. Tenho direcionado meus estudos para especialização nessa linguagem de programação e embora não possua experiência em projetos reais em breve estarei incluindo alguns trabalhos, ainda em desenvolvimento, aqui nesse mesmo site.
